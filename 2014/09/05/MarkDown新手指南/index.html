Markdown新手指南

By MiShuai

发表于 2014-09-05

1. 标题

在Markdown中，要想文本显示为标题，只需要在文本前面加上’#’即可，
Markdown总共有6级标题，在文本前面加几个‘#’就是几级标题，例如在文
本前加 ‘#’ 就是一级标题，加 ‘##’ 就是二级标题，后面以此类推。
从一级标题到六级标题字体逐渐减小。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

2. 列表

在Markdown中，在文字前加上-
即可，例如：

无序列表：

- 文本1
- 文本2
- 文本3

有序列表：

1. 文本1
2. 文本2
3. 文本3

效果为：

3. 链接和图片

在Markdown中，插入链接格式为： [显示文本] (链接地址)

例如：

[百度](www.baidu.com)

在Markdown中，插入图片格式为：！[ ] (图片链接地址)

例如：

![](https://www.baidu.com/img/bdlogo.png)

4. 引用

在Markdown中只需在希望引用的文字前加上 > 就好了，例如：

> 一盏灯， 一片昏黄； 一简书， 一杯淡茶。 守着那一份淡定， 品读属于自己的寂寞。 保持淡定， 才能欣赏到最美丽的风景！ 保持淡定， 人生从此不再寂寞。

5. 粗体和斜体

在Markdown中，斜体的格式为： 

*需要斜体的文字*正常字体的文字

粗体的格式为:

**需要粗体的文字**不需要粗体的文字
