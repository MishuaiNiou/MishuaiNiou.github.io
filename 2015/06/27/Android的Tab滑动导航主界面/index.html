Android的Tab滑动导航主界面

By MiShuai

发表于 2015-06-27

Android

上次使用Fragment实现了Android的Tab导航主界面，但是没有左右滑动导航的效果，这次我会在上次的基础上添加左右滑动导航的效果。

动态效果图如下：

实现的思路是使用ViewPager+Fragment。

首先来看看什么是ViewPager。

官方文档给出的是：

Layout manager that allows the user to flip left and right through pages of data. You supply an implementation of a PagerAdapter to generate the pages that the view shows.

ViewPager is most often used in conjunction with Fragment, which is a convenient way to supply and manage the lifecycle of each page. There are standard adapters implemented for using fragments with the ViewPager, which cover the most common use cases. These are FragmentPagerAdapter and FragmentStatePagerAdapter; each of these classes have simple code showing how to build a full user interface with them.

ViewPager是一个允许用户在一组数据页面间左右滑动的布局管理器。通过实现PagerAdapter可以生成视图显示的界面。

ViewPager总是和Fragment连在一起使用，这是一个提供和管理每个页面生命周期的方便的方法。将ViewPager和Fragments一起使用有涵盖了大多数普通用例的标准的适配器实现，它们是FragmentPagerAdapter和FragmentStatePagerAdapter，其中每个类都有简单的代码展示了如何构建一个完整的用户界面。

总的来说，ViewPager就是一个装载页面的容器，有左右滑动的特性，它和Fragment一起使用时，通过设置和实现FragmentPagerAdapter或FragmentStatePagerAdapter就可以对填充在ViewPager中的Fragments进行管理和数据控制。

主界面的布局和上次Android的Tab导航主界面中的布局一样，滑动的实现思路是：将4个Fragment放入到ArrayList中，ViewPager设置适配器为继承了FragmentPagerAdapter的适配器，里面重写了getItem()和getCount()方法可以获取list集合中fragments的数量和位置。MainActivity实现OnPageChangeListener来监听页面滑动的变化，必须实现其中的onPageScrollStateChanged()、onPageScrolled()、onPageSelected()方法，在onPageSelected()方法中设置Tab点击颜色为亮色就可以了。至于点击跳转对应页面可以调用ViewPager的setCurrentItem()方法，需要注意的是setCurrentItem()默认一个参数的方法会有在点击跳转时会有滑动的动画效果，如果想去除动画效果，可以调用2个参数的setCurrentItem()方法并将第二个参数值设为false。
