Hibernate初体验

By MiShuai

发表于 2015-09-12

ORM（Object Relation Mapping），对象/关系数据库映射，完成了面向对象编程语言到关系数据库的映射，将一张数据表映射成了持久化类，将数据表的行映射成对象，将数据表的列（字段）映射成了对象的属性。

Hibernate是一个面向Java环境的ORM（对象/关系数据库映射）框架，用来把对象模型表示的对象映射到基于SQL的关系模型数据结构中去,可以使程序员通过面向对象的方法来操纵数据库。

使用Hibernate的步骤如下:

  1. 去hibernate官网下载hibernate，我下载的是hibernate4.2。创建一个Java工程，将所需jar包加入libs文件夹下。

  2. 创建一个User类，添加3个私有属性：username，password和age，并生成对应的get和set方法，代码如下：

    public class User {

    private String username;
    private String password;
    private int age;

    public User() {
        super();
        // TODO Auto-generated constructor stub
    }

    public String getUsername() {
        return username;
    }

    public void setUsername(String username) {
        this.username = username;
    }

    public String getPassword() {
        return password;
    }

    public void setPassword(String password) {
        this.password = password;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    @Override
    public String toString() {
        return "User [username=" + username + ", password=" + password + ", age=" + age + "]";
    }

}

  3. 生成User.hbm.xml文件，用来配置持久化类到数据库的映射，如下：


    <id name="username" type="java.lang.String">
    <column name="USERNAME" />
    <generator class="assigned" />
</id>
<property name="password" type="java.lang.String">
    <column name="PASSWORD" />
</property>
<property name="age" type="int">
    <column name="AGE" />
</property>


  4. 创建hibernate.cfg.xml文件，在文件中配置数据库的基本信息。如下：


    <!-- 配置连接数据库的基本信息 -->
<property name="connection.username">root</property>
<property name="connection.password"></property>
<property name="connection.driver_class">com.mysql.jdbc.Driver</property>
<property name="connection.url">jdbc:mysql://localhost:3307/hibernate</property>

<!-- 配置hibernate使用的数据库方言 -->
<property name="dialect">org.hibernate.dialect.MySQLDialect</property>

<!-- 是否在控制台打印sql语句 -->
<property name="show_sql">true</property>

<!-- 是否格式化sql语句 -->
<property name="format_sql">true</property>

<!-- 指定自动生成数据表的策略 -->
<property name="hbm2ddl.auto">update</property>

<!-- 指定关联的 .hbm.xml文件-->
<mapping resource="com/hibernate/bean/User.hbm.xml" />


  5. 创建测试类Test，代码如下：

    public class Test {
    @org.junit.Test
    public void test() {
        // 创建一个 SessionFactory 对象
        SessionFactory sessionFactory = null;

        // 创建 Configuration 对象: 对应 hibernate 的基本配置信息和 对象关系映射信息
        Configuration configuration = new Configuration().configure();

        // 4.0 之前这样创建:
        // sessionFactory = configuration.buildSessionFactory();

        // 创建一个 ServiceRegistry 对象: hibernate 4.x 新添加的对象
        // hibernate 的任何配置和服务都需要在该对象中注册后才能有效.
        ServiceRegistry serviceRegistry = new ServiceRegistryBuilder()
                        .applySettings(configuration.getProperties())
                        .buildServiceRegistry();

        sessionFactory = configuration.buildSessionFactory(serviceRegistry);

        // 创建一个 Session 对象
        Session session = sessionFactory.openSession();

        // 开启事务
        Transaction transaction = session.beginTransaction();

        // 执行保存操作
        User user = new User();
        user.setUsername("Jack");
        user.setPassword("123");
        user.setAge(20);
        session.save(user);

        // 提交事务
        transaction.commit();

        // 关闭 Session
        session.close();

        // 关闭 SessionFactory 对象
        sessionFactory.close();
    }
}

Run As->JUnit Test成功就可以看到数据库中生成的user表中插入一条记录了。
